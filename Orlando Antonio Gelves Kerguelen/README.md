# ACERCA DE MI #
Uno de mis fragmentos de codigos favoritos es el algoritmo de *Fibonacci recursivamente*, lo adjunto escrito en *Python*

```
def fib(n):
    if n == 0:
        return 0
    if n == 1:
        return 1
    return fib(n-1) + fib(n-2)
```
